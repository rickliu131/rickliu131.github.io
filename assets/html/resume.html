Projects

  • Generating Live Visual Descriptions for Environment Understanding
    - Contributed to the design and development of a system that generates personalized visual descriptions, based on the specified intent and real-time visual/audio context of the Blind and Visually Impaired (BVI) users.
    - Implemented essential modules for processing visual data. Researched algorithms for extracting keyframes to identify major scene transitions in the user’s environment, enabling the system to generate more adaptable visual descriptions.
    - Assisted in conducting user study with six participants to evaluate the system.
    Link to ACM Digital Library
  • Non-Visual Image Editing with Natural Language Input and Verification
    - Developed an image editing tool, enabling BVI users to perform common object-level edits using natural language instructions, and verify the edits through Large Multimodal Model-generated visual descriptions and edit summaries.
    - Ran two user study sessions. Analyzed data to evaluate user confidence in performing editing tasks.
    Link to ACM Digital Library
  • Speech-Controlled Hand Interaction in Virtual Environments
    - Implemented modules to parse speech instructions into VR space-recognizable commands using ASR and LLM models.
    - Collected data from 14 study participants. Analyzed speech-based hand interaction tasks success rates and causes of errors.
  • Assistive Tool for Creating Immersive Audio Storytelling Experiences
    - Led a four-month project brainstorming effort, resulting in developing and evaluating an assistive tool that streamlines the production of immersive, sound effects-augmented storytelling audio, that requires minimal expertise and labor effort.
    - Researched 30+ podcasts/audiobooks to explore existing audio storytelling experiences and sound augmentation techniques.
  • LLM-Based Argumentative Essay Writing Tutor
    - Ran studies with experts and novices to investigate the cognitive model of argumentative writing and common challenges.
    - Built the tutoring system based on the study's discoveries, cognitive tutor principles, and the Toulmin argument model.
